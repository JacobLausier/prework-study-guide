HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • measurments of objects are in px for pixel
  • style css is used to make the website more presentable

Git

  • Git cd (enter name)- gets us to the file directory we need to be in
  • Git pull- pulls from the repository on gitbash
  • Git push- pushes all staged changes to github repository
  • Git checkout- will put us into whatever branch we need to go to

Javascript

  • var - lets us create a variable with a given name. Can also be used with an array.
  • console.log -lets us edit the inspect console on our web application.
